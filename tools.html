My Web Authoring Software

The software I am using for my code is Notepad++. This is a basic software that is meant to be for typing notes and/or basic code of many kinds. You can make an HTML or CSS file, and type working code in it, but it has no tools that change the view from normal to design to code. There is also no easy insert tabs of any kind, basically the only way to insert is through typing.

An authoring tool we also barely got to use was Adobe Dreamweaver, but it was still used to learn how to use a web authoring software.
