Web Accessability Checklist
User Characteristics Accessible Design Tips
Unable to see Code all images with alternate text
Unable to perceive colors Avoid using color alone to convey information
Unable to use mouse Be sure all website features can be accessed using a keyboard
Unable to hear Add captions to multimedia pieces
Prone to having seizures Avoid content that flashes
Easily distractible Keep the design simple

Accessibility Review of the Bordentown Regional High School Home Page

When put into an evaluation tool, the school's website had...

  1. 0 direct errors in code or CSS
  2. 12 alerts involving suspicious alternate text and redundant links
  3. 39 contrast errors with poor color options

The above alerts and contrast errors can be fixed by typing more reliable alternate text pieces and choosing a better contrasting color scheme(even though it is our school colors).
